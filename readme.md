<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* install Godot

### Run Game

1. Download the Artefacts for your OS from the latest [Action](https://github.com/janekx21/lizard-world/actions)
2. Unpack zipfile
3. Run lizard-world-client.x86_64
